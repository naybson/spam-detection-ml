# Spam Detection – ML Assignment

NLP text classification project using the SMS Spam Collection dataset from Kaggle.

## Dataset
[SMS Spam Collection Dataset](https://www.kaggle.com/datasets/team-ai/spam-text-message-classification) – Kaggle

## Assignment type
- **Task:** Text analysis (NLP)
- **Learning type:** Binary classification
- **Algorithm:** Naive Bayes / SVM (see notebook)
